** Installation

#+BEGIN_SRC shell
$ git clone https://github.com/hlissner/doom-emacs.git .emacs.d
$ cd ~/.emacs.d
$ git branch develop origin/develop
$ git checkout develop
$ cd
$ git clone https://github.com/jwintz/doom.git .doom.d
$ cd ~/.doom.d
$ ln -s init.doom.el ~/.emacs.d/init.el
$ cd ~/.emacs.d
$ make
$ make doctor
#+END_SRC
